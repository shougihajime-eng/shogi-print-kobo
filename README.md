# はじめ先生の将棋プリント工房 (shogi-print-kobo)

将棋教室向けの A4 プリント生成ツール。ブラウザだけで動き、データは端末内に保存される。

## できること

- **詰将棋プリント** — 1手・3手・5手 の詰将棋を最大4問まで選び、A4 1枚にまとめて印刷
- **駒の動かし方** — 8種類の駒の動きを図と説明文で解説したプリント
- **将棋の反則** — 二歩・打ち歩詰めなど主要な反則ルール一覧プリント
- **玉の囲い** — 矢倉・美濃・舟・穴熊 の基本形プリント
- **盤面エディタ** — 9x9 盤に駒をクリック配置、持駒も管理して詰将棋を登録
- **問題ライブラリ** — 登録済み問題をフィルタ・JSON エクスポート

データは `localStorage` に保存されるため、同じブラウザ・同じ端末で永続化される。

## 動かし方

```bash
npm install
npm run dev
```

http://localhost:3000 を開く。

## 印刷

各プリント画面で「プリントを生成 →」を押すとプレビューが出る。「🖨 印刷する」（または Ctrl+P）で A4 印刷ダイアログが開く。
